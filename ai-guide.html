Guide complet Fableris

Tout ce que tu dois savoir pour jouer, créer et soumettre tes histoires.

Les pages de l'application
📚

Bibliothèque

La page principale. Toutes les histoires disponibles sont regroupées par pack. Les histoires gratuites sont accessibles directement. Les histoires payantes affichent un cadenas 🔒 — elles se débloquent via un code ou un achat. Appuie sur une histoire pour voir sa description et la commencer.

🎮

Le moteur de jeu

Le cœur de Fableris. Tu lis les scènes et tu fais tes choix. En haut : la barre de progression et tes stats (audace, prudence, savoir...). En bas : les choix disponibles. Certains choix 🔒 sont verrouillés si tes stats ne sont pas assez élevées. Tape sur l'écran pour afficher/masquer l'interface et entrer en mode immersif.

📊

Ma Progression

Retrouve toutes tes parties en cours et terminées. Pour chaque histoire tu vois combien de fins tu as découvertes sur le total possible. Tu peux reprendre une partie là où tu l'as laissée ou recommencer depuis le début pour explorer d'autres chemins.

📰

Actualités

Les dernières nouvelles de Fableris — nouvelles histoires, mises à jour, événements spéciaux. C'est ici que tu seras informé des prochaines sorties et des codes promotionnels.

✍

Soumettre une histoire

Tu as une idée d'aventure ? Crée-la dans l'éditeur StoryForge et envoie-la nous. Les meilleures histoires sont publiées officiellement dans l'application. Tu recevras le badge Auteur et des avantages exclusifs.

Packs et déblocage
🆓

Contenu gratuit

Plusieurs histoires sont entièrement gratuites dès l'installation — dont le Pack Enfants complet. Aucun compte requis, aucune connexion nécessaire. Tout fonctionne hors ligne.

🔑

Codes de déblocage

Certains packs se débloquent avec un code. Dans Options → Entrer un code, saisis le code reçu par e-mail ou lors d'un événement. Le contenu est débloqué définitivement sur ton appareil.

🔒

Histoires Via Portail

Certaines histoires affichent le badge "Via Portail". Elles font partie d'un pack premium et nécessitent un accès spécial. Les testeurs officiels reçoivent un code pour y accéder gratuitement.

Stats et fins alternatives

Comment fonctionnent les stats ? Chaque choix que tu fais peut augmenter une stat — Audace, Prudence, Savoir ou d'autres selon l'histoire. Ces stats influencent les choix disponibles et les fins possibles. Un joueur audacieux verra d'autres options qu'un joueur prudent.

🎯

Fins multiples

La plupart des histoires ont plusieurs fins possibles. Pour toutes les découvrir, rejoue l'histoire en faisant des choix différents. Ta progression est sauvegardée automatiquement — tu peux revenir à n'importe quel moment.

🔒

Choix conditionnels

Un choix grisé avec 🔒 indique qu'il nécessite un certain niveau de stat. Par exemple "Intervenir avec autorité" peut nécessiter Audace ≥ 30. Rejoue l'histoire en favorisant cette stat pour débloquer ces chemins cachés.

Éditeur StoryForge — Vue d'ensemble

StoryForge est l'éditeur visuel intégré à Fableris. Il permet de créer des histoires complètes sans toucher au JSON. L'arbre visuel montre toutes tes scènes et leurs connexions en temps réel.

🌳

L'arbre visuel

Chaque bulle est une scène. Les flèches montrent les chemins possibles. Les flèches orange pointillées avec 🔒 indiquent des choix conditionnels. La bulle verte est la scène de départ, les bulles rouges sont les fins. Pince pour zoomer, glisse pour naviguer.

✏

Éditer une scène

Tape sur une scène pour l'éditer. Tu peux modifier le titre, le lieu (affiché en haut de la scène), le narrateur (texte en italique avant le texte principal), le texte principal, la transition visuelle et l'illustration emoji. Chaque modification est sauvegardée automatiquement.

→

Les choix

Chaque scène peut avoir plusieurs choix. Un choix a un texte, une destination (vers quelle scène il mène) et des effets sur les stats. Tu peux ajouter une condition 🔒 — le choix n'apparaîtra que si le joueur a assez de points dans une stat. Le bouton Lier permet de connecter visuellement deux scènes.

⚙

Les stats

Définis les stats de ton histoire dans ⚙ Stats — courage, lucidité, chaleur... Chaque stat a un nom, une couleur et une valeur de départ. Les choix peuvent les modifier avec des valeurs positives ou négatives. Elles permettent des fins différentes selon le profil du joueur.

▶

Tester l'histoire

Le bouton ▶ Tester ouvre ton histoire dans le vrai moteur de jeu. Tu peux simuler différents niveaux de stats pour vérifier que tes fins conditionnelles fonctionnent correctement. Reviens à l'éditeur avec le bouton retour.

Tous les boutons du menu ⋯

Sur mobile, le menu ⋯ en haut à droite donne accès à toutes les actions. Sur tablette/PC, elles sont visibles directement dans la barre.

⚙

🎨

Choisis parmi des thèmes prédéfinis ou personnalise les couleurs de fond, bordures et accents de ton histoire.
⊞

▶

Lance ton histoire dans le moteur de jeu pour la tester comme un vrai joueur. Simule les stats pour tester les chemins conditionnels.
↑

↓

✎

✉

🧒

👤
Demander le prénom
Si activé, le jeu demande son prénom au joueur avant de commencer. Le prénom peut apparaître dans les textes de l'histoire.

Badge santé ✓ — En haut à droite de l'éditeur, ce badge indique l'état de ton histoire. Vert = tout va bien. Jaune = avertissements. Rouge = erreurs bloquantes (scène sans destination, choix cassé, pas de scène de départ). Clique dessus pour voir la liste détaillée.

🔍 Recherche & Navigation — Le bouton loupe en bas à gauche de l'arbre permet de chercher une scène par nom. La minimap en bas à droite donne une vue d'ensemble et permet de naviguer rapidement dans les grandes histoires. Les boutons +/− et ◎ contrôlent le zoom et recentrent la vue.

Créer avec l'IA
01

Décris ton idée à l'IA

Ouvre ChatGPT, Claude ou Gemini. Décris l'univers de ton histoire, les personnages, l'ambiance, les choix que tu veux proposer. Pas besoin de connaître le format technique.

02

Utilise le prompt Fableris

Copie le prompt ci-dessous et colle-le dans ta conversation avec l'IA. Il explique exactement le format JSON attendu par le moteur Fableris.

03

Importe dans l'éditeur

L'IA génère un fichier JSON complet. Ouvre StoryForge, utilise "Importer JSON" dans le menu ⋯ et ta scène s'affiche directement dans l'arbre visuel.

04

Affine et envoie

Retouche les textes, ajuste les choix, teste l'aperçu. Quand tu es satisfait, exporte et envoie-nous ton histoire depuis l'éditeur.

Conseil : Pour de meilleurs résultats, fournis d'abord un exemple de JSON existant à l'IA en lui demandant d'analyser la structure, puis demande-lui de créer une nouvelle histoire dans le même format. Tu peux télécharger un exemple depuis l'éditeur via "Exporter JSON".

Le prompt Fableris

Copie ce prompt et colle-le dans ta conversation avec l'IA, suivi de la description de ton histoire.

Prompt à copier
Analyse attentivement le JSON fourni. Tu dois comprendre précisément : - la structure des scènes, - le système de choix, - les transitions, - les fins, - les variables, - les effets, - les IDs, - le style d'écriture, - le format exact attendu. Ensuite, génère une NOUVELLE histoire complète parfaitement compatible avec cette structure JSON. IMPORTANT : - respecte exactement le même format, - ne casse aucun lien entre les scènes, - chaque choix doit pointer vers une scène existante, - garde des IDs cohérents et uniques, - conserve la compatibilité avec le moteur narratif, - ajoute des descriptions longues et immersives, - privilégie les ambiances fortes, - les personnages récurrents, - les chemins annexes, - les fins alternatives, - les scènes absurdes ou marquantes si le ton de l'histoire s'y prête. L'histoire doit être directement exploitable sans modification technique. Le style doit ressembler à une vraie histoire interactive moderne, avec : - narration immersive, - dialogues naturels, - scènes mémorables, - choix intéressants, - et une structure claire pour le joueur. Ne donne aucune explication. Retourne uniquement le JSON complet final.
Structure d'une scène

Pour aider l'IA à comprendre le format, voici la structure minimale d'une scène :

"ma_scene": { "id": "ma_scene", "title": "Titre de la scène", "location": "LIEU — CONTEXTE", "narrator": "Texte en italique avant la scène.", "text": "Le texte principal de la scène...", "isStart": false, "isEnd": false, "transition": "fade", "choices": [ { "text": "Ce que le joueur peut choisir", "vers": "id_scene_suivante", "effects": [ { "stat": "courage", "val": 10 } ], "clickEffect": "pulse" } ] }
